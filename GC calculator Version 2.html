GC Calculator Version 2

Calculates methane from GC concentration, vial weights, sludge density, brine volume, vial volume and poison volume.

Upload GC Input File

Use a clean Excel sheet with the raw input columns only. The calculator will generate the calculation columns.

Required input columns:
Sampling date, Date_GC, Sample_name, Sampling Technique, Lab_GC, Concent_CH4 (%), Wi (g), Wf (g), Density (g/ml), Brine_vol (mL), V_vial (mL), V_poison (mL)

Optional input column: T_sample (°C). If missing, the calculator assumes 21°C.

Calculated by the tool:
V_headspace (mL), V_sludge (mL), Cg, Total_P, Pg, Molar_water, T_reference, T_sample (K), Kh, Molecular_mass_CH4, Standard volume at NTP, C_A, C_AH, TC.
Calculation summary:
Liquid volume from weight = (Wf - Wi) / Density
V_headspace = Brine_vol + Liquid volume from weight
V_sludge = V_vial - V_poison - V_headspace
Cg = Concent_CH4 (%) / 100
Pg = Cg × Total_P
C_A = (Molar_water × Pg × Molecular_mass_CH4 × 1000) / Kh
C_AH = (Cg × Molecular_mass_CH4 × 1000 / Standard volume at NTP) × (V_headspace / V_sludge) × (T_reference / T_sample_K)
TC = C_A + C_AH
