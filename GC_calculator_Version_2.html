Upload GC Input File

Upload the clean GC sheet with the exact columns below.

Expected uploaded columns:
Sampling date, Date_GC, Sample_name, Sampling Technique, Lab_GC, Concent_CH4 (%), Wi (g), Wf (g), Density (g/ml), Brine_vol (mL), V_vial (mL), V_headspace (mL), V_poison (mL), V_sludge (mL), T_sample (°C)

Important: this version does not recalculate V_headspace or V_sludge. It uses the values already present in the uploaded Excel file.
Calculated by the tool:
Cg = Concent_CH4 (%) × 10^-2
Total_P = 1 atm
Pg = Cg × Total_P
Molar_water = 55.5 mol/L
T_reference = 273.15 K
T_sample (K) = T_reference + T_sample (°C)
Kh = 1 / EXP(-338.217 + (13282.1 / T_sample_K) + 51.9144 × LN(T_sample_K) - 0.0425831 × T_sample_K)
Molecular_mass_CH4 = 16.04 g/mol
Standard volume at NTP = 22.4 L/mol
C_A = Molar_water × (Pg / Kh) × Molecular_mass_CH4 × 10^3
C_AH = (V_headspace / V_sludge) × Cg × ((Molecular_mass_CH4 / Standard volume at NTP) × (T_reference / T_sample_K)) × 10^3
TC = C_A + C_AH
